<task type="auto">
  <name>Task 1: Install D3 packages and create network page with graph data and visualization</name>
  <files>site/package.json, site/src/pages/network.astro, site/src/components/Header.astro</files>
  <action>
    1. Install D3 individual packages (NOT the full d3 bundle):
       ```
       cd site && npm install d3-force d3-selection d3-zoom d3-drag
       ```

    2. Add "Network" link to Header.astro nav alongside Timeline and Search:
       ```html
       <a href="/network" class="text-gray-600 hover:text-gray-900 transition-colors">Network</a>
       ```

    3. Create `site/src/pages/network.astro` with two parts:

    **Part A -- Build-time frontmatter (server-side graph data computation):**
    - Import `getCollection` from `astro:content` and `BaseLayout`
    - Fetch all posts via `getCollection("posts")`
    - Build `nodes` array: each post becomes `{ id: post.id, title: post.data.title, theme: post.data.theme, concepts: post.data.concepts, url: "/posts/" + post.id }`
    - Build `links` array: for each pair of posts (i, j where j > i), find shared concepts via `filter(c => posts[j].data.concepts.includes(c))`. If shared.length > 0, push `{ source: posts[i].id, target: posts[j].id, sharedConcepts: shared }`
    - Serialize to JSON: `const graphData = JSON.stringify({ nodes, links })`

    **Part B -- HTML structure:**
    - Wrap in `<BaseLayout title="Concept Network - IAIFI Research Blog">`
    - Header section: h1 "Concept Network", subtitle "Explore connections between papers through shared concepts. Nodes are papers, edges are shared concepts."
    - Instructions text (small, gray): "Drag nodes to rearrange. Scroll to zoom. Click a node to read the paper."
    - Container div: `id="network-container"`, full width, `min-h-[500px] h-[70vh]`, white bg, rounded with shadow and border, `overflow-hidden`, `data-graph={graphData}`
    - Below the graph: a legend showing the 4 IAIFI theme colors as colored circles with labels (Foundational AI = blue-500, Theoretical Physics = violet-500, Experimental Physics = amber-500, Astrophysics = emerald-500), laid out as a horizontal flex row
    - Handle empty state: if nodes.length < 2, show a message "Not enough papers to visualize connections. Add more papers through the admin interface." instead of the graph container

    **Part C -- Client-side D3 script (use regular `<script>` tag, NOT `is:inline`, so Vite bundles the D3 imports):**

    Import from D3 individual packages:
    ```javascript
    import { forceSimulation, forceLink, forceManyBody, forceCenter, forceCollide } from "d3-force";
    import { select } from "d3-selection";
    import { zoom, zoomIdentity } from "d3-zoom";
    import { drag } from "d3-drag";
    ```

    On DOMContentLoaded:
    - Get `#network-container`, parse `dataset.graph` JSON into `{ nodes, links }`
    - Read container dimensions: `width = container.clientWidth`, `height = container.clientHeight`
    - Define theme color map matching global.css values:
      ```
      "Foundational AI": "#3b82f6"
      "Theoretical Physics": "#8b5cf6"
      "Experimental Physics": "#f59e0b"
      "Astrophysics": "#10b981"
      ```

    - Create SVG via d3 `select(container).append("svg")` with width, height, and `viewBox` attributes
    - Create a `<g>` group inside SVG for zoom/pan transforms

    - Render links as `<line>` elements in a group with `stroke: "#d1d5db"`, `stroke-opacity: 0.6`, stroke-width scaled by `Math.sqrt(d.sharedConcepts.length)`

    - Render nodes as `<circle>` elements in a group:
      - Radius: 8px base, or scale by connection count if desired (keep simple: 8px)
      - Fill: theme color from map, fallback "#6b7280"
      - Stroke: "#fff", stroke-width: 1.5
      - Cursor: pointer

    - Create HTML tooltip div (NOT SVG title -- richer display):
      - Append to body: `select("body").append("div")`
      - Classes: `"absolute hidden bg-gray-900 text-white text-sm px-3 py-2 rounded-lg shadow-lg pointer-events-none z-50 max-w-xs"`
      - Style position absolute
      - On mouseover: show tooltip with `<strong>${d.title}</strong><br/><span style="opacity:0.7">${d.theme}</span><br/><span style="opacity:0.5">${d.concepts.length} concepts</span>`
      - On mousemove: position at `event.pageX + 12`, `event.pageY - 12`
      - On mouseout: hide tooltip

    - Set up force simulation:
      ```javascript
      const simulation = forceSimulation(nodes)
        .force("link", forceLink(links).id(d => d.id).distance(d => Math.max(50, 150 - d.sharedConcepts.length * 20)).strength(d => Math.min(1, d.sharedConcepts.length * 0.15)))
        .force("charge", forceManyBody().strength(-200).distanceMax(400))
        .force("center", forceCenter(width / 2, height / 2).strength(0.05))
        .force("collide", forceCollide(12).strength(0.7))
      ```
      CRITICAL: `.id(d => d.id)` on forceLink -- without this, links match by array index and connect wrong nodes.

    - On each tick, update line x1/y1/x2/y2 and circle cx/cy positions

    - Set up zoom + pan on SVG element:
      ```javascript
      svg.call(zoom().scaleExtent([0.3, 5]).on("zoom", (event) => { g.attr("transform", event.transform); }));
      ```

    - Set up drag on node circles (apply BEFORE click handler):
      - Use `isDragging` flag pattern to prevent click from firing after drag
      - on("start"): set isDragging=false, if !event.active reheat simulation with alphaTarget(0.3).restart(), pin d.fx=d.x, d.fy=d.y
      - on("drag"): set isDragging=true, update d.fx=event.x, d.fy=event.y
      - on("end"): if !event.active set alphaTarget(0), release d.fx=null, d.fy=null

    - Click handler on nodes: `if (!isDragging) window.location.href = d.url;`

    - Hover highlight: on mouseover also increase node stroke-width to 3 and reduce opacity of non-connected nodes/links to 0.15. On mouseout restore all. This makes the graph explorable by highlighting a node's neighborhood.
