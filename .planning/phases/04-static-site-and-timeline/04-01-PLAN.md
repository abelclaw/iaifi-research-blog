<task type="auto">
  <name>Task 1: Python export script for approved blog posts</name>
  <files>pipeline/export.py</files>
  <action>
Create `pipeline/export.py` -- a synchronous Python script (not async, since it runs standalone before the Astro build) that:

1. Connects to the SQLite database at `data/papers.db` (using `sqlite3`, not `aiosqlite` -- this is a build-time script, not an async server handler).
2. Queries all approved blog posts joined with their paper metadata:
   ```sql
   SELECT bp.*, p.authors, p.abstract, p.iaifi_category,
          p.published, p.arxiv_id, p.pdf_url
   FROM blog_posts bp
   JOIN papers p ON bp.paper_arxiv_id = p.arxiv_id
   WHERE bp.status = 'approved'
   ORDER BY p.published DESC
   ```
3. For each approved post:
   - Queries concepts: `SELECT name, description, relevance FROM concepts WHERE paper_arxiv_id = ? ORDER BY relevance DESC`
   - Queries figures: `SELECT figure_path FROM figures WHERE paper_arxiv_id = ? ORDER BY sort_order`
   - Copies figure files from `data/figures/` to `site/public/figures/` (create directory if needed)
   - Writes a markdown file to `site/src/content/posts/{slug}.md` with YAML frontmatter containing:
     - `title` (string), `arxivId` (string), `authors` (list of strings, parsed from JSON), `abstract` (string), `theme` (string from iaifi_category), `published` (ISO date string), `arxivUrl` (constructed as `https://arxiv.org/abs/{arxiv_id}`), `pdfUrl` (from pdf_url or fallback to `https://arxiv.org/pdf/{arxiv_id}`), `concepts` (list of concept name strings), `figures` (list of root-relative URLs like `/figures/filename.png`), `wordCount` (integer)
   - The markdown body is the blog post `content` field from the database

4. Before writing, clears any existing `.md` files in `site/src/content/posts/` to prevent stale content from rejected or deleted posts.

5. Add `pyyaml` to pyproject.toml dependencies. Use `yaml.dump(frontmatter, default_flow_style=False)` for YAML serialization.

6. Include a `__main__` block so it can be run as `python -m pipeline.export`:
   ```python
   if __name__ == "__main__":
       db_path = settings.DB_PATH
       site_dir = "site"
       count = export_approved_posts(db_path, site_dir)
       print(f"Exported {count} approved posts to {site_dir}/src/content/posts/")
   ```

7. The function signature: `def export_approved_posts(db_path: str, site_dir: str) -> int`

Note: Use `sqlite3` (sync stdlib) not `aiosqlite`. This script runs at build time, not inside the async FastAPI server.

  </verify>

  <name>Task 2: Scaffold Astro project with Tailwind CSS v4 and content collection</name>
  <files>site/astro.config.mjs, site/src/content.config.ts, site/src/styles/global.css, site/package.json, site/tsconfig.json</files>
  <action>
1. Create the Astro project manually (do NOT use `npm create astro` interactive wizard). Create the `site/` directory and write files directly:

   **site/package.json:**
   ```json
   {
     "name": "iaifi-blog",
     "type": "module",
     "version": "0.0.1",
     "scripts": {
       "dev": "astro dev",
       "build:astro": "astro build",
       "build:search": "npx pagefind --site dist",
       "build": "npm-run-all -s build:astro build:search",
       "preview": "astro preview"
     }
   }
   ```

2. Install dependencies from `site/` directory:
   ```bash
   cd site && npm install astro @astrojs/mdx @astrojs/sitemap tailwindcss @tailwindcss/vite && npm install -D pagefind npm-run-all
   ```

3. **site/tsconfig.json:** Use `"extends": "astro/tsconfigs/strict"`

4. **site/astro.config.mjs:** Configure with:
   - `@astrojs/mdx` and `@astrojs/sitemap` integrations
   - `@tailwindcss/vite` as a Vite plugin (NOT `@astrojs/tailwind` -- that is deprecated for v4)
   - `site: "https://iaifi.github.io"` (placeholder, configurable later)
   - Do NOT set `output: 'server'` -- this is a fully static site

5. **site/src/content.config.ts:** Define the posts collection with:
   - `glob({ pattern: "**/*.md", base: "./src/content/posts" })` loader
   - Zod schema matching the export script frontmatter:
     - `title: z.string()`
     - `arxivId: z.string()`
     - `authors: z.array(z.string())`
     - `abstract: z.string()`
     - `theme: z.enum(["Foundational AI", "Theoretical Physics", "Experimental Physics", "Astrophysics"])`
     - `published: z.coerce.date()`
     - `arxivUrl: z.string().url()`
     - `pdfUrl: z.string().url()`
     - `concepts: z.array(z.string())`
     - `figures: z.array(z.string()).default([])`
     - `wordCount: z.number()`

6. **site/src/styles/global.css:** Create with:

   ```css
     --color-theme-theory: #8b5cf6;
     --color-theme-experiment: #f59e0b;
     --color-theme-astro: #10b981;
   }
   ```

7. Create placeholder directories:

   - `site/src/pages/` (empty, populated in plan 02)
   - `site/src/components/` (empty, populated in plan 02)
